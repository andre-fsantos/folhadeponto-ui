# Get started üí°
To start off you need to setup your development environment

## Docker setup üêã
Once you already have docker installed you'll need to run only one command

```bash
# run with sudo if your docker executable does not have sudo privileges
docker compose up -d # this will run in detach mode, logs will be hidden
```

Then you can check the project running at `http://localhost:5173`

```bash
# run with sudo if your docker executable does not have sudo privileges
docker compose logs
```


## Manual setup

After clone the project all you need to do is run the install script.
This command will install all dependencies of the project, without it you won't be able to run the project
```bash
# please use npm
npm install
```

Next, you need to up the project, so run the dev script
```bash
# this will run vite dev bin
npm run dev
```

Then you can check the project running at `http://localhost:5173`
